# Email Spam Detection using Machine Learning

## Project Description
This project detects whether an email is Spam or Ham using Machine Learning techniques.
It helps in filtering unwanted emails automatically.

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas

## Project Features
- Upload email dataset
- Train the model
- Classify emails as Spam or Not Spam

## Project Structure
- Dataset files – spam_ham_dataset.csv
- Model files – rf.txt, tfidf.txt
- Python scripts for training and prediction

## How to Run
1. Install Python
2. Install required libraries
3. Run manage.py or run.bat
4. Run the script to classify emails as Spam or Ham

## Output
The system classifies emails as Spam or Ham based on trained data.
