# Email Spam Detection using Machine Learning

## Project Description
This project detects whether an email is Spam or Ham using Machine Learning techniques.
It helps in filtering unwanted emails automatically.

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas
- Django
- HTML, CSS

## Project Features
- Upload email dataset
- Train the model
- Classify emails as Spam or Not Spam
- Display results through web interface

## Project Structure
- Dataset files – spam_ham_dataset.csv
- Model files – rf.txt, tfidf.txt
- Django app files – HTML, Python files
- manage.py – Project execution file

## How to Run
1. Install Python
2. Install required libraries
3. Run manage.py or run.bat
4. Open browser and test spam detection

## Output
The system classifies emails as Spam or Ham
